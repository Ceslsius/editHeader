# editHeader
edit request header plugin 修改请求header头浏览器插件

1.修改background.js中targetUrl和headerObj为你需要的目标url和header头参数

2.打开浏览器插件管理后台，并进入开发者模式

3.点击 加载已解压的扩展程序，选择当前文件夹

4.刷新页面，就能看到生效
